## Setup（セットアップ）

1. [Releases](https://github.com/{user}/{repo}/releases) から `.alfredworkflow` ファイルをダブルクリックしてインストール。
2. Alfred Preferences でこのワークフローを開き、**Configure Workflow** をクリック。
3. 必要な設定を入力する。

## Usage（使い方）

| トリガー | 説明 |
|---|---|
| `wf <query>` | 検索（デフォルト） |
| `wf open <name>` | ショートカットを開く |
| `wf config` | 設定の確認 / リセット |
| `wf help` | コマンド一覧を表示 |
